# 空气质量传感器电路原理

```
                    +3.3V
                      │
                      ├──────────────────┐
                      │                  │
                     ┌┴┐                ┌┴┐
                     │ │ 10kΩ           │ │ 10kΩ
                     │ │                │ │
                     └┬┘                └┬┘
                      │                  │
     ESP32 GPIO21 ────┴──── I²C SDA     │ I²C SCL
                      │                  │
                      ├──── SGP30 SDA    ├──── SGP30 SCL
                      │                  │
                      │    PMS5003       │
                      │    ┌─────────────┤
                      │    │             │
                     ┌┴┐  ┌┴┐          ┌┴┐
                     │ │  │ │           │ │
                     └┬┘  └┬┘          └┬┘
                      │    │             │
                     GND  GND          GND

                    +5V
                     │
                     ├──── DHT22 VCC
                     │
                    ┌┴┐
                    │ │ 4.7kΩ (上拉)
                    │ │
                    └┬┘
                     │
    ESP32 GPIO4 ─────┴──── DHT22 DATA

                     +3.3V
                      │
                     ┌┴┐
                     │ │
                     └┬┘
                      │
    ESP32 GPIO5 ──────┴──── OLED SDA
    ESP32 GPIO18 ──────── OLED SCL
```

## PCB 设计提示

1. PMS5003 的 UART 走线尽量短 (< 10cm)
2. I²C 总线上拉电阻: SDA/SCL 各 10kΩ
3. 电源去耦: 每个传感器 VCC 引脚附近放置 100nF MLCC
4. 电源布线: 12V 和 5V 线路分开，避免数字噪声耦合
5. 在 KiCad 下设计，原理图放在 `hardware/pcb/` 目录
