# 安全测试协议

> 所有测试方法设计为**可在家完成**，无需专业实验室。

## 1. 气密性测试

### 烟雾法 (定性)
- **工具:** 线香/蚊香/发烟器
- **步骤:**
  1. 在外罩内点燃线香
  2. 关闭外罩所有开口
  3. 用手电筒照射所有接缝
  4. 观察烟雾是否从接缝渗出
  5. 用密封条密封有泄漏处，重复测试

### 压差法 (半定量)
- **工具:** 压差计 (¥30-50, 淘宝可购)
- **步骤:**
  1. 启动排风扇至最大转速
  2. 在外罩内外各放一个压差计探头
  3. 记录稳定后压差 (目标: ≥ 20 Pa)
  4. < 10 Pa 表示密封不足

## 2. 过滤效率测试

### PM2.5 过滤效率
- **工具:** 两台 PMS5003 (一台参考)
- **步骤:**
  1. 在外罩外放置第一台 PMS5003 (环境基线)
  2. 在外罩排风口放置第二台 PMS5003
  3. 打印机打印 ABS，记录 30 分钟数据
  4. 过滤效率 = (1 - 出口PM/环境PM) × 100%
  5. 目标: 出口 PM2.5 < 50 µg/m³，效率 > 80%

### VOC 削减率
- **工具:** SGP30 传感器 (内外各一个)
- **步骤:**
  1. 记录打印前内外 TVOC 基线
  2. 打印开始后每 5 分钟记录读数
  3. 计算削减率
  4. 目标: 出口 TVOC < 250 ppb

## 3. 温升测试

- 打印机运行 1 小时 PLA 打印
- 记录外罩内部 vs 外部温差
- 目标: 内部温升 ≤ 8°C

## 4. 噪音测试

- 使用手机 App (如 "Decibel X")
- 在距外罩 1m 处测量
- 排风扇运行状态
- 目标: ≤ 45 dB(A)
