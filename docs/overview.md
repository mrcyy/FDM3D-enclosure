# 项目概览

## 设计目标

1. **健康优先** — 将桌面打印的健康风险从"看不见"变为"可量化、可控制"
2. **模块化** — 核心机械/电子/软件组件相互独立，可单独升级或替换
3. **低成本** — 总物料成本控制在 $100-150，远低于商用方案
4. **可验证** — 所有防护效果均可通过公开方法自行测试验证
5. **可复现** — 提供完整的设计文件、BOM、装配指南、固件源码

## 适用打印机

| 类型 | 适配性 | 说明 |
|------|--------|------|
| FDM < 300×300mm | ✅ 完美 | 默认设计尺寸 |
| FDM < 400×400mm | ✅ 可扩展 | 调整面板尺寸即可 |
| 树脂/SLA | ✅ 需加强 | 建议增加活性炭层厚度 |
| 大型FDM (>500mm) | ⚠️ 参考 | 框架需加强，可作为设计参考 |

## 安全层次

```
┌────────────────────────────────┐
│    第1层: 物理隔离 (外罩本体)     │  ← 降低背景暴露
├────────────────────────────────┤
│    第2层: 主动排风 (负压系统)     │  ← 快速排出污染物
├────────────────────────────────┤
│    第3层: 过滤 (HEPA + 活性炭)   │  ← 捕获颗粒物和VOC
├────────────────────────────────┤
│    第4层: 实时监测 (空气质量)     │  ← 量化防护效果
└────────────────────────────────┘
```

## 设计约束

- 内部尺寸: ≥ 400×400×450mm (适用于绝大多数桌面机)
- 面板材料: 5mm 透明亚克力 / PETG
- 框架: 2020 铝型材
- 密封: EPDM 橡胶条
- 工作温升: ≤ 8°C (相对环境)
- 噪音: ≤ 45dB @ 1m
