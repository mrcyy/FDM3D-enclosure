# 组装指南

## 准备工作

- [ ] 打印所有结构件 (查看 `models/` 目录)
- [ ] 切割亚克力面板 (按 BOM 尺寸)
- [ ] 采购全部电子元件 (参考 BOM)
- [ ] 准备工具: 内六角扳手(4mm)、螺丝刀、电烙铁、3D打印机

---

## 第一步: 框架搭建

1. 用角码将 8 根 500mm 铝型材竖立为四角立柱
2. 接入 8 根 450mm 横梁，形成长方体框架
3. 用水平仪校准，预紧所有螺栓
4. 框架外部尺寸: 500×500×500mm

> 💡 **提示:** 可在型材槽内预埋 T 型螺母，方便后续安装附件

---

## 第二步: 面板安装

1. 在面板边缘粘贴 EPDM 密封条
2. 先安装底板 (可打胶固定或螺丝锁紧)
3. 安装侧板 — 从内部用角码或L型支架固定
4. 安装前后面板 (其中一面做开门设计)
5. 顶板暂不安装 (方便走线)

### 开门改造 (推荐)
- 在前面板两侧安装合页
- 面板中间切分: 2/3 为活动门，1/3 固定
- 安装拉手和 (可选) 磁吸扣

---

## 第三步: 风道与过滤系统

```
              ┌─────────────┐
  外排 ←──────┤ 排风扇 (+)  │ ←── HEPA+活性炭
              └──────┬──────┘
                     │ 风管
              ┌──────┴──────┐
              │   外罩内部   │
              └─────────────┘
```

1. 在背板或侧板开 120mm 圆孔 (排风口)
2. 安装风扇固定座 (3D打印件)
3. 安装 HEPA → 活性炭 → 预过滤 → 排风扇 (从外到内)
4. 在底部或对面侧板开 50mm 被动进气孔
5. (可选) 安装进气风扇 + 可调百叶

---

## 第四步: 电子系统

### 接线框图

```
        ┌──────────────────────┐
        │    12V/3A 电源       │
        └──┬─────┬────────────┘
           │     │
     LM2596 │     └──→ 排风扇
      (5V)  │
        │   │
    ┌───┘   │
    ▼       ▼
  ESP32    Arduino
  ┌────┐   ┌──────┐
  │GPIO│   │PWM   │←── 传感器数据(可选)
  │UART│←──┤I²C   │
  └────┘   └──────┘
```

### ESP32 引脚分配 (Air Quality Monitor)

| 引脚 | 连接 | 说明 |
|------|------|------|
| GPIO16 (RX2) | PMS5003 TX | PM2.5 数据输入 |
| GPIO17 (TX2) | PMS5003 RX | 预留 |
| GPIO21 (SDA) | SGP30 SDA | I²C 数据 |
| GPIO22 (SCL) | SGP30 SCL | I²C 时钟 |
| GPIO4 | DHT22 DATA | 温湿度 |
| GPIO5 | OLED SDA | 显示 |
| GPIO18 | OLED SCL | 显示时钟 |
| GPIO13 | 蜂鸣器 | 报警提示 |

### Arduino/协处理器引脚分配 (Fan Control)

| 引脚 | 连接 | 说明 |
|------|------|------|
| GPIO9 (PWM) | 排风扇 PWM | 转速控制 |
| GPIO10 | 进气风扇 PWM | 可选 |
| A0 | 传感器反馈 | 接收ESP32控制信号 |

## 第五步: 固件烧录

1. 安装 PlatformIO IDE (VS Code 扩展)
2. 进入 `firmware/air-quality-monitor/`
3. 编辑 `include/config.h` 设置WiFi信息
4. 编译并上传:
   ```bash
   pio run --target upload
   ```
5. 风扇固件同理:
   ```bash
   cd firmware/fan-control
   pio run --target upload
   ```

---

## 第六步: 校准与测试

1. 气密性测试:
   - 关闭外罩，点燃一根香/蚊香在内
   - 检查是否有烟雾渗出
   - 用密封条/胶带修补漏点

2. 传感器校准:
   - 在洁净空气中开机运行 30 分钟 (SGP30 自校准)
   - 对比 PMS5003 读数是否合理 (室外 ~30-50 µg/m³)

3. 排风效率测试:
   - 打印机运行 ABS/ASA
   - 观察监测读数 30 分钟
   - 排风状态下 PM2.5 应 < 50 µg/m³
