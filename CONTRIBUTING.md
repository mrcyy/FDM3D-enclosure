# 贡献指南

感谢您对 FDM3D-Enclosure 感兴趣！

## 如何贡献

### 提交 Issue

- 🐛 **Bug 报告** — 包括硬件/固件版本、现象、复现步骤
- 💡 **功能请求** — 描述你需要但缺失的功能
- 🔬 **测试结果** — 分享你的外罩测试数据

### 提交 PR

1. Fork 本仓库
2. 创建功能分支: `git checkout -b feat/your-feature`
3. 提交更改
4. 推送: `git push origin feat/your-feature`
5. 提交 Pull Request

### 代码规范

- 固件: Arduino C++ (遵循 Google C++ Style)
- 软件: Node.js (遵循 StandardJS)
- 文档: Markdown (中文 + 英文)

## 测试贡献

如果你搭建了外罩并进行了测试，请分享：
- 测试环境描述
- 传感器读数 (CSV/JSON)
- 改良建议

## 行为准则

- 尊重每一位贡献者
- 关注技术讨论，避免人身攻击
- 欢迎各水平的参与者和建议
